计算机网络: HTTP请求/响应头
发表于2023-06-20|计算机网络
请求头Referer请求头字段 Referer 是 HTTP 请求头的一部分，用于指示请求的来源页面的 URL。它通常由浏览器自动添加到请求头中，告知服务器当前请求是从哪个页面链接跳转而来的。 Referer 字段的值是一个 URL 字符串，表示当前请求所在页面的 URL。它可以提供给服务器一些有用的信息，例如： 流量来源分析：服务器可以根据 Referer 字段分析流量来源，了解访问者是通过哪个页面链接或搜索引擎访问当前页面的。 防盗链（Hotlinking）检测：网站管理员可以通过检查 Referer 字段来判断资源的请求是否来自本站，用于防止盗链和保护资源。 安全验证：某些情况下，服务器可能会使用 Referer 字段来进行安全验证，确保请求来源于可信的页面。
