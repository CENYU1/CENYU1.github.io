attribute 和 property 的区别

attribute 是在 HTML 标签中定义的，而 property 则是在 DOM 中定义的，两者都有可能引起 DOM 重新渲染

  1. 定义方式不同：属性是定义在 HTML 标签上的，特性是定义在 DOM 对象上的
  2. 数据类型不同：属性的值只能是字符串，而特性的值可以是字符串、数字、布尔值等多种数据类型
  3. 可读性不同：属性的值通过 getAttribute 方法读取，而特性的值可以直接读取
  4. 可写性不同：属性的值可以通过 setAttribute 方法修改，而特性的值可以直接修改
